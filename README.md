# Dulce Manía - v0.1.0

This is the initial version of the "Dulce Manía" web application, built with Firebase Studio.

## Tech Stack

This project is a modern web application built with a powerful and scalable technology stack. Below are the main technologies and libraries used:

### Framework & Libraries

*   **Next.js:** `15.5.9` - A React framework for production-grade applications. It enables features like server-side rendering and generating static websites.
*   **React:** `19.2.1` - A JavaScript library for building user interfaces.
*   **TypeScript:** `5` - A typed superset of JavaScript that compiles to plain JavaScript, enhancing code quality and maintainability.

### UI & Styling

*   **Tailwind CSS:** `3.4.1` - A utility-first CSS framework for rapidly building custom user interfaces.
*   **ShadCN/UI:** A collection of re-usable UI components.
*   **Lucide React:** `0.475.0` - A comprehensive library of simply designed icons.

### Generative AI

*   **Genkit:** `1.20.0` - A framework from Google for building AI-powered features and applications.

### Forms & Validation

*   **React Hook Form:** `7.54.2` - A library for flexible and extensible forms with easy-to-use validation.
*   **Zod:** `3.24.2` - A TypeScript-first schema declaration and validation library.
