# **App Name**: Crazy Bakery

## Core Features:

- Image Carousel: Displays a rotating image carousel on the homepage to showcase the products that have been created.
- Contact Form: Offers a "Contact" section with a form for users to submit inquiries and contact information.
- User Authentication: Implements a secure user authentication system (login/registration) to manage user accounts and orders. This system must have its own authentication and be federated by Google.
- Order Request: Allows users to request dessert orders and store them in the Spring backend for processing. (Coming Soon).

## Style Guidelines:

- Colors should be in pastel shades.
- Primary color: Vibrant coral (#FF7F50) to evoke a sense of energy and action.
- Background color: Light beige (#F5F5DC), providing a neutral backdrop that complements the primary color.
- Accent color: Teal (#008080), used for calls to action and highlights, creating a visual contrast.
- Body font: 'PT Sans', sans-serif font providing readability and a modern feel.
- Headline font: 'Playfair', a modern serif to create an elegant style
- Use simple and clear icons to represent different categories and actions within the application.
- Incorporate subtle transitions and animations for a smooth user experience when navigating between pages or submitting forms.
